This WordPress plugin allows your users to conveniently create gift certificates from your site.


== Description ==

Gift Certificate Creator WordPress plugin allows you to manage gift certificates on your website. In a convenient front-end UI provided by this plugin, your site visitors can enter the amount and user details. On form submission, the user details will be sent to the administrator. Also, the administrator can view the list of all the certificate requests.


== Installation ==

= Follow the steps to install the plugin: =

1. Navigate to Add New Plugin page within your WordPress
2. Search for Current Promotions Display Plugin
3. Click Install Now link on the plugin and follow the prompts
4. Activate the plugin through the 'Plugins' menu in WordPress

Or

1. You can also download the plugin from the following location:
http://bobcares.com/gift-certificate-creator.html
2. Upload plugin directory to the /wp-content/plugins/ directory
3. Activate the plugin through the 'Plugins' menu in WordPress

On successful activation, a menu 'Gift Certificates' will be displayed on the 'admin dashboard'.
To display the certificate order form, you can add the shortcode '[gift_certificate_form]' in page/post content from admin side.
